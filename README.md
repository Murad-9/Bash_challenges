# Bash Script Challenges

This repository contains a set of Bash scripting challenges i completed to strengthen my Linux and shell scripting fundamentals.

Each challenge focuses on a specific concept such as arithmetic operations, file handling, permissions, and automation.

## Challenges
- Challenge 1: Arithmetic Calculator
- Challenge 2: Directory and file automation
- Challenge 3: File checker with permissions
- Challenge 4: backup script for text files

All scripts were written and tested using VS code.
