# Challenge 1: Arithmetic Calculator

This Bash script prompts the user to enter two numbers and perform basic arithmetic operations on them.

## What the script does
- Reads two numbers from user input
- Calculates:
  - Addition
  - Subtraction
  - Multiplication
  - Division
- Handles division by zero Safely

## Concepts practiced
- User input with 'read'
- Arithmetic expansion '$(( ))'
- Conditional statements ( if / else )
- Output formatting with 'echo'

## Notes
This challenge helped me understand how Bash handles variables, arithmetic and basic input validation
